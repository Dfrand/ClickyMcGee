# Clicky McGee
Exciting memory improving game utilizing React.js!

 ***
## Getting Started

 - Clone repo.
 - Run `npm install` in command line.
 - Run `yarn start` in command line.

***
## Technologies Used:

 * React.js
 * Node.js
 * CSS3
 * HTML5

***
##  Installs

### React
 - `npm install react`

### React-Dom
 - `npm install react-dom`
 
### React-Scripts
 - `npm install react-scripts`

### SweetAlert
 - `npm install sweetalert`
